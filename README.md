# NutriLens

AI-powered food and object scanner built using SwiftUI + CoreML.

## Features
- Real-time object detection
- Camera integration
- Nutrition insights
- Offline AI inference

## Tech Stack
- SwiftUI
- Vision Framework
- CoreML
- AVFoundation
- MVVM

## Architecture
NutriLens/
├── App/
├── Features/
│   ├── Camera/
│   ├── Detection/

├── Core/

│   ├── ML/
│   ├── Storage/
│   └── Utilities/
├── Shared/
│   ├── Components/
│   └── Extensions/
└── Resources/
